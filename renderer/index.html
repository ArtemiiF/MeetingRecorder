Настройки и готовность

        
      


      

        
Обновления 

        

        

          
          
          
        

        
Установка обновлений доступна только в собранном приложении — в режиме разработки только проверка.

        
После обновления macOS может заново спросить разрешения (ad-hoc подпись меняется между сборками).

      


      

        
Перед началом работы

        
    
          
  1. Запусти LM Studio и загрузи модель — нужна для сводок, заголовков и PARA-разбора (сервер на :1234).
    2. 
          
  2. Подними лимит токенов в LM Studio: Context Length ≥ ~20k и output ≥ 16000 — иначе PARA-разбор обрывается на размышлениях.
    3. 
          
  3. Дай доступ к микрофону и записи экрана (для системного звука) в Системных настройках macOS.
    4. 
          
  4. ffmpeg ставится автоматически вместе с бэкендом (кнопка «⬇ Установить бэкенд» ниже); для разработки его можно поставить отдельно: brew install ffmpeg.
    5. 
          
  5. Опционально: HF-токен для диаризации (имена спикеров). Без него — спикеры по таймкодам.
    6. 
          
  6. Модель Whisper скачается автоматически при первой транскрипции (~1.5 ГБ).
    7. 
        

      


      

        
Оформление

        
      


      

        
Личные данные

        
        
        
Нужен для диаризации (разделения записи на спикеров); дальше LLM пробует подставить их реальные имена по тексту. Где взять токен? — создай токен типа Read и прими условия моделей pyannote.

        
⚠️ Системный keychain недоступен — токен хранится в обратимом виде (base64), не зашифрован.

      


      

        
Модель для быстрых задач

        
        
Механические LLM-вызовы (коррекция словаря, заголовок, предложения терминов) пойдут в эту модель вместо загруженной по умолчанию — например, google/gemma-3-4b. Пусто — использовать загруженную модель. Список подсказок — из моделей, доступных сейчас в LM Studio.

      


      

        
Модель для основных задач

        
        
Основные LLM-вызовы (сводка встречи, чат по заметкам, PARA-разбор, извлечение действий, определение спикеров) пойдут в эту модель вместо загруженной по умолчанию. Пусто — использовать загруженную модель (сводка останется на модели с рассуждением, как сегодня). Список подсказок — из моделей, доступных сейчас в LM Studio.

      


      

        
Куда сохранять

        

          
          
        

        
Если настроен PARA-vault — путь подстраивается под него автоматически (папка «Meetings» внутри vault). «Изменить» задаёт свой путь и отключает автоподстройку.
